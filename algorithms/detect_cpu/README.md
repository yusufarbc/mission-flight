# detect_cpu

CPU-only detection experiments used to validate edge cases without GPU acceleration.

## Files

- `canny_edge.py` - Canny edge detection example.
- `detection_demo.py` - Simple detection demo for quick tweaks.
- `legacy_ieee_anka.py`, `legacy_demo.mp4` - Legacy scripts and footage kept for reference.

## Usage

```powershell
pip install opencv-python numpy
python canny_edge.py
```

Replace `canny_edge.py` with any script above to test a different routine.
